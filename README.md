# Pizza Sales Analytics Dashboard

An end-to-end Business Intelligence project developed using Power BI and MySQL to analyze pizza sales performance and generate actionable business insights through interactive dashboards and data visualization.

## Project Overview

This project focuses on transforming raw transactional sales data into a fully interactive analytics dashboard. The dashboard helps track business performance, monitor KPIs, identify sales trends, and analyze customer ordering behavior across multiple dimensions.

The project involved importing and cleaning raw datasets, transforming date and categorical fields using Power Query, validating KPI calculations through SQL queries in MySQL, and designing a user-friendly dashboard in Power BI.

## Objectives

* Analyze overall sales performance
* Monitor key business KPIs
* Identify top-performing products
* Understand customer ordering patterns
* Create interactive reports for decision-making

## Key Features

### KPI Tracking

* Total Revenue
* Total Orders
* Total Pizzas Sold
* Average Order Value
* Average Pizzas per Order

### Sales Analysis

* Monthly revenue trends
* Daily order distribution
* Sales by pizza category
* Sales by pizza size
* Top and bottom-performing pizzas

### Interactive Dashboarding

* Dynamic filtering and slicers
* Drill-down analysis
* Interactive visuals and reporting
* Time-based performance analysis
